# Скажи и забудь

Голосовой прототип для быстрых бытовых напоминаний: сроков, событий и будильника.

## Что умеет

- сохранять фразы вроде `Молоко до 5 июня`;
- понимать `сегодня`, `завтра`, `послезавтра`;
- понимать интервалы: `через 30 минут`, `через месяц`;
- сохранять события со временем: `Звонок маме сегодня в 8 вечера`;
- ставить будильник: `Будильник завтра на 9 утра`;
- показывать ближайшие сроки списком;
- удалять отдельные записи или очищать все.

## Статус

Это тестовый прототип. Уведомления работают только пока страница открыта.
