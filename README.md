# скажи и забудь

Простые голосовые напоминания на русском: скажи, **что** и **когда** напомнить, — и забудь.

Работает и как веб-приложение (PWA), и как нативное приложение для Android (через Capacitor).

## Что умеет

- **Голосом** — нажми «старт» и скажи фразу в формате «что и когда». Например:
  - «молоко завтра», «завтра позвонить маме»
  - «стрижка 1 марта в 5 часов», «таблетка через 30 минут»
  - «встреча в полвосьмого», «без пятнадцати пять»
- **Вручную** — то же самое можно напечатать в поле «или напиши».
- Понимает «сегодня / завтра / послезавтра», дни недели, точные даты, «через N минут/часов/дней…», время суток и разговорное время.
- **Изменение** записи голосом или вручную: «изменить дату на завтра», «измени на 5 минут позже».
- **Поиск** по названию, дате или периоду.
- **Напоминания**: в Android-приложении приходят как системные уведомления — даже когда приложение закрыто. В веб-версии (PWA) — пока вкладка открыта.

## Технологии

- Чистый ванильный JavaScript, без фреймворков.
- Нативная обёртка — [Capacitor](https://capacitorjs.com/) (Android): нативное распознавание речи и системные уведомления.

## Разработка

Веб-код (он же источник и для приложения) лежит в корне: `index.html`, `script.js`, `styles/`, `service-worker.js`.

Чтобы правки попали в Android-приложение:

    npm install          # один раз
    npm run build:web    # собрать веб-файлы в папку www
    npx cap sync         # перенести в Android-проект

Затем собрать APK/AAB в Android Studio (папка `android`).

Проверка разбора голосовых фраз (без телефона):

    node scripts/parse-test.js
